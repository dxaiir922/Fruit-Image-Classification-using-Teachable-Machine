# Fruit-Image-Classification-using-Teachable-Machine
## Project Overview

This project demonstrates an image classification model built using Google Teachable Machine. The model classifies fruit images into three categories:

- Apple
- Banana
- Strawberry

The trained model was exported in TensorFlow (Keras) format and used in a Python script to classify new images.

## Objectives

- Train an image recognition model using Google Teachable Machine.
- Export the trained model in TensorFlow (Keras) format.
- Load the model using Python.
- Predict the class of an input image.
- Display the prediction result.

## Dataset

The dataset consists of three classes:

- Apple
- Banana
- Strawberry

Each class contains five training images.

## Tools Used

- Google Teachable Machine
- TensorFlow / Keras
- Python
- NumPy
- Pillow (PIL)

## Project Files

- `keras_model.h5` – Trained model
- `labels.txt` – Class labels
- `Untitled14.ipynb` – Prediction script

## Workflow

1. Create the classes in Teachable Machine.
2. Upload training images.
3. Train the model.
4. Export the model in TensorFlow (Keras) format.
5. Run the Python script to classify a new image.

## Example Output

**Input Image:** Apple

**Prediction:** Apple

**Confidence:** 99%

## Conclusion

This project demonstrates a simple image classification system using Google Teachable Machine and Python. The trained model successfully classifies fruit images with high accuracy.
